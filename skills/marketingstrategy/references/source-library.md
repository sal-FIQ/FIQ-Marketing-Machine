# Source library for examples, blogs, newsletters, substack, and medium

Use this file when you want fresh patterns, tactical examples, or operator-style explanations that go beyond core frameworks.

## How to use this file

- Prefer the **research and book-based sources** in `marketing-knowledge-pack.md` for the strategic spine.
- Use this file for examples, tactical nuance, phrasing, and operator patterns.
- Treat practitioner blogs, newsletters, Substack, and Medium pieces as **supporting evidence**, not final authority.
- Give more weight to sources that show real reasoning, clear examples, or documented outcomes.

## 6) How to combine source types

### Best blend for a strong strategy
Use this order of precedence:
1. user inputs and company-specific research
2. research-backed sources and foundational books
3. practitioner sites and institute libraries
4. Substack, newsletters, and Medium for examples or operator tone

### Example of good use
- use LinkedIn B2B Institute and Ehrenberg-Bass to justify why the plan includes both demand creation and demand capture
- use April Dunford to build positioning and messaging architecture
- use Refine Labs and a strong ABM article to shape how the team operationalizes pipeline creation
- use a Substack or Medium piece only to enrich examples, sequencing, or phrasing

### Example of bad use
- copying a Medium article’s channel list into every strategy
- recommending Substack as a default channel for every company
- treating one anecdotal ABM case study as proof that ABM is always the right motion

## 7) Final caution

This file is for enrichment, not worship.

Use it to make the strategy feel sharper, more current, and more operator-aware. But keep the plan grounded in:
- the company’s actual sales motion
- buyer behavior
- market maturity
- credible revenue logic
